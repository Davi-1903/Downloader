## Como executar

1. **Clone e acesse o repositório**

    ```bash
    git clone https://github.com/Davi-1903/Downloader.git
    cd Downloader
    ```

2. **Instale a aplicação**

    ```bash
    uv tool install .
    ```

3. **Execute em qualquer lugar**

    ```bash
    downloader
    ```
